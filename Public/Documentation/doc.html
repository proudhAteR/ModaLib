Section Item 1

Modals

How it works

Our modals are built using HTML, CSS and JavaScript. They are rendered at top level using a container at position: fixed having a the actual modal inside. The container is used as a filter for the background of the modal.

You'll need few things to personalize your modals:

  • A data-trigger to define what kind of modal you are using

    There's 3 kinds of modals:

    1. Alert modals
    2. Confirm modals
    3. Custom modals

  • An id to link the button to its modal.
  • A data-animation to define the animation to use when the modal appears.

    The animations you can use are:

    1. fade
    2. slide

    This value is set to fade by default.

  • A data-action for the main button of the modal and a data-second fro the second button of the modal.
  • A data-options to set the what the buttons in your modal display as a message.

    This value is set to OK by default.

  • A data-title to set the what the title of the modal.
  • A data-say to set the what the message of the modal.
  • A data-color to change the color of the action button in the modal (the button on the right of the modal) This attribute has a default value for each type of modal.

    You just have to use the first part of the color name. (alert, confirm, estella).

You just need to link the script and the style file of the library in your html file.
